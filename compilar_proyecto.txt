Pasos para compilar el proyecto HotelManagementC

1. Navega al directorio del proyecto:
   cd ~/HotelManagementC

2. Configura el proyecto con CMake:
   cmake -S . -B build

3. Compila el proyecto:
   cmake --build build

4. Ejecuta el programa:
   cd bin
   ./HotelManagementC.exe
